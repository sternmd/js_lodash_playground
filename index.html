Reasons why I like lodash:

  • Many functions work on collections: map or reduce whole objects chaining: _(elements).map(...).map(...).filter().filter(...).value() Is lazy evaluated, and all those maps and filters are reduced (shortcut fusion) so the data is only being traversed once.
  • flatten and flattenDeep
  • clone and cloneDeep to prevent accidental mutation
...And basically all the other methods that can be found in the docs :p
